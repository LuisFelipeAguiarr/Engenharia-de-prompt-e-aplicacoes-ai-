# Engenharia-de-prompt-e-aplicacoes-ai-
    ( Apresenta a atividade) 
 Este repositório é para a disciplina de Prompt e Aplicações em Inteligência Artificial. Explora técnicas e metodologias para projetar, otimizar e avaliar interações com modelos de linguagem e sistemas de IA generativa.O foco está em transformar ideias em instruções eficazes que resultem em respostas úteis, criativas e confiáveis.
